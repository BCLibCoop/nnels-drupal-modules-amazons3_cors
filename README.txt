Version compatibility
--------------------------------------------------------------------------------
This module requires the AmazonS3 module (https://www.drupal.org/project/amazons3).
And tracks the version numbering scheme of that module.

Example:

+----------+---------------+
| AmazonS3 | AmazonS3 CORS |
+----------+---------------+
| 7.x-1.x  | 7.x-1.x       |
+----------+---------------+
| 7.x-2.x  | 7.x-2.x       |
+----------+---------------+

Used this project for reference to get some things working.
https://github.com/dgoodlad/s3-photo-upload-demo/
